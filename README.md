# Sleek — Car Rental Landing Page

A modern, animated car rental landing page built with React and Vite. Sleek features smooth scroll-driven animations, a clean component architecture, and a fully responsive design showcasing car categories, top fleet picks, brands, testimonials, and more.

---

## Requirements

- [Node.js](https://nodejs.org/) v18+
- [Yarn](https://yarnpkg.com/)

---

## Getting Started

**1. Install dependencies**

```bash
yarn install
```

**2. Start the dev server**

```bash
yarn dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser and you're good to go.

---

## Tech Stack

- **React 18** — UI framework
- **Vite** — Lightning-fast dev server and bundler
- **Tailwind CSS** — Utility-first styling
- **Motion / GSAP** — Scroll animations
- **Radix UI** — Accessible UI primitives
- **React Router** — Client-side routing
